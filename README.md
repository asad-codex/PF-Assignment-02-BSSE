# C++ University Console Applications Collection 🚀

This repository contains a collection of console-based C++ applications developed using **procedural programming** principles as part of university coursework. 

---

## 📂 Included Projects

| # | Project Name | Description | Source File |
|---|---|---|---|
| 1 | **LESCO Electricity Bill System** | Calculates tiered monthly utility bills, taxes, fixed charges, and new connection estimates for LESCO. | [`Electricity Bill LESCO.cpp`](./Electricity%20Bill%20LESCO.cpp) |
| 2 | **Grocery Mart Billing System** | Manages cart items, calculates itemized bills, sales tax, membership discounts, and loyalty points. | [`Grocery Mart Billing System.cpp`](./Grocery%20Mart%20Billing%20System.cpp) |
| 3 | **Online Shopping System** | E-commerce checkout system handling product carts, city-based delivery fees, GST, and customer discounts. | [`Online Shopping System.cpp`](./Online%20Shopping%20System.cpp) |
| 4 | **Personal Student Diary** | Secure authentication-based diary system to create, read, edit, delete, and search personal journal entries. | [`Personal Student Diary.cpp`](./Personal%20Student%20Diary.cpp) |
| 5 | **Restaurant Management System** | Handles dine-in/takeaway food orders, service charges, GST, tier-based discounts, and delivery eligibility. | [`Resturant Management System.cpp`](./Resturant%20Management%20System.cpp) |
| 6 | **Social Media Management System** | Computes digital marketing campaign costs based on platforms, post types, ad budgets, and agency fees. | [`Social Media Management System.cpp`](./Social%20Media%20Management%20System.cpp) |

---

## 🛠️ Prerequisites & Compilation

All programs are written in standard **C++** and can be compiled using any C++ compiler (like GCC, MinGW, Clang, or MSVC).
